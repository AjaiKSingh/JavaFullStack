# JavaFullStack

**Backend (Spring Boot):**
	Java, Spring Boot, Spring Data JPA, RESTful API: Exposes endpoints for communication with the frontend.
**Frontend (React):**
	React, State Management, React Router , Fetch
**Database:**
	Database MySQL: Stores employee information persistently.
